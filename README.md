# My first webpage 

This is a basic frontend webpage with a clean layout.

This web page consists of text which is aligned in the center for a cleaner look along with a list of my hobbies which I made using 'ul list' which means unordered list.
It has background colour of cyan to look attractive which I made using the attribut bg color.
It consists of images which are related to my Hobbies mentioned earlier
A form for user to input his name, surname and hobbies along with textbox for the same
To make sure that webpage looks clean i used br tag wherever neccesary to skip a line.
Another table made specifying the top three car brands in my opinion
